Python Data Analysis Project

Project Overview

Python data analysis project using Pandas, Matplotlib, and Jupyter Notebook to explore airline data, perform data cleaning, create visualizations, and extract business insights.

Tools Used

  • Python
  • Pandas
  • Matplotlib
  • Data Cleaning
  • Data Visualization

Business Questions

  • How are passenger ages distributed across the dataset?
  • What is the distribution of bookings between Economy and Business classes?
  • Which destinations generated the highest total revenue?
  • Which cancellation reasons resulted in the highest average refund amounts?
  • How are ticket prices distributed across all bookings?
