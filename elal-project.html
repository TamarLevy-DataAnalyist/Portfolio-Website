EL AL Flight Analysis Dashboard

Project Overview

Power BI dashboard analyzing EL AL flight data during 2024–2025, including revenue, passengers, cancellations, flight activity, and occupancy trends.

Tools Used

  • Power BI
  • Power Query
  • DAX
  • Data Modeling
  • Data Visualization

Business Questions

  • How did passenger volumes and flight activity change over time?
  • Which destinations generated the highest revenue?
  • What cancellation trends appeared during the analyzed period?
  • How did occupancy rates vary by destination and season?
