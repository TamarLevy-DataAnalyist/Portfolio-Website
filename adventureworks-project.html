Advanced SQL Business Analysis

Project Overview

Advanced SQL analysis project focused on extracting business insights from a relational sales database. The project includes revenue analysis, customer ranking, product profitability, supplier-product relationships, geographic revenue analysis, cumulative monthly totals, pivot reporting, and customer churn identification.

Tools Used

  • SQL Server
  • T-SQL
  • CTE
  • Window Functions
  • Pivot Tables
  • Aggregations
  • Business Analysis

Business Questions

  • How did yearly income change over time, and what was the growth rate?
  • Who were the top customers by revenue in each year and quarter?
  • Which products generated the highest profit?
  • Which suppliers provide which products?
  • Which customers and regions generated the highest revenue?
  • How did monthly income accumulate throughout each year?
  • How can order volume be compared across years using pivot reporting?
  • Which customers may be at risk of churn based on order behavior?

SQL Techniques Demonstrated

  • Common Table Expressions for structured query logic.
  • Window functions such as LAG, ROW_NUMBER, DENSE_RANK, and cumulative SUM.
  • Aggregations using SUM, COUNT, AVG, and GROUP BY.
  • Ranking customers and products based on business performance.
  • Pivot reporting for comparing monthly order activity across years.
  • Customer churn logic based on order frequency and inactivity.

Project Files

The project includes SQL scripts containing advanced business queries for sales, customers, products, suppliers, revenue trends, and customer activity analysis.
