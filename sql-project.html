Retail Management SQL Database Project

Project Overview

SQL project focused on designing and creating a relational database for a retail management system. The database includes employees, customers, product categories, products, and orders, with relationships implemented through primary keys and foreign keys.

Tools Used

  • SQL
  • SQL Server
  • Relational Database Design
  • Primary Keys and Foreign Keys
  • Data Modeling

Project Scope

  • Created a relational database named RetailManagement.
  • Designed five connected tables: Employees, Customers, Categories, Products, and Orders.
  • Defined primary keys using IDENTITY columns.
  • Created relationships between tables using foreign keys.
  • Inserted sample data to simulate a retail business environment.

Project Files

The project includes SQL scripts for creating the database structure, defining tables, building relationships, and inserting sample data.
